# HR Analytics Dashboard

Interactive Power BI dashboard visualizing workforce metrics including attrition rate, department headcount, and employee satisfaction.

## Tools Used
- **Power BI** — Dashboard design, DAX measures, slicers
- **Microsoft Excel** — Data preprocessing using pivot tables and conditional formatting before Power BI ingestion

## Features
- DAX measures for workforce metrics
- Slicers for dynamic filtering by department, age group, and job role
- Visualizations covering attrition rate, department headcount, and employee satisfaction

## Files
- `HR_Analytics.xlsx` — Source data
- `HR Analytics Dashboard.pbix` — Power BI dashboard file
